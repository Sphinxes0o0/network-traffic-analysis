## 1. 研究背景

### 1.1 问题

加密流量分类面临挑战：
- 传统方法依赖 payload
- 需要新特征表示

### 1.2 创新点

FlowSpectrum：将流量转换为频谱表示。

---

## 2. 方法论

### 2.1 FlowSpectrum 表示

```
流量序列 → 频谱转换 → 一维坐标表示

核心思想：
- 将包长度序列看作信号
- 使用 FFT 转换到频域
- 提取频谱特征
```

### 2.2 深度学习分类

```
频谱特征 → CNN → 分类输出
```
