---

## 1. 研究背景

### 1.1 问题

- 单一方法不足
- 内部威胁复杂多样
- 误报率高

### 1.2 创新点

多层混合框架结合滥用检测和异常检测。

---

## 2. 方法论

### 2.1 混合框架

```
┌─────────────────────────────────────────────────────────────┐
│                 多层内部威胁检测框架                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   层 1: 滥用检测                                             │
│   └── 已知恶意模式                                           │
│                                                              │
│   层 2: 异常检测                                             │
│   └── 偏离正常行为                                           │
│                                                              │
│   层 3: 集成输出                                             │
│   └── 综合判断                                               │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

---

## 3. 主要贡献

1. 多层检测架构
2. 滥用 + 异常混合
3. 降低误报率

---

## 4. 结论

多层框架比单一方法更有效。
