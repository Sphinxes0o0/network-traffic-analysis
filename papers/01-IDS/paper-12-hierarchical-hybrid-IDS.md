---

## 1. 研究背景

### 1.1 问题

- 单一检测方法不足
- 多层防护需求
- 降低误报率

### 1.2 创新点

层次化混合检测架构。

---

## 2. 方法论

### 2.1 层次化架构

```
┌─────────────────────────────────────────────────────────────┐
│                    层次化 IDS 架构                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   第1层: 快速过滤                                           │
│   └── 规则匹配                                             │
│                                                              │
│   第2层: 异常检测                                           │
│   └── 统计异常                                             │
│                                                              │
│   第3层: 深度检测                                           │
│   └── 内容分析                                             │
│                                                              │
│   输出: 分层告警                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

---

## 3. 主要贡献

1. 多层过滤架构
2. 减少误报
3. 分层处理

---

## 4. 结论

层次化架构提升 IDS 的准确性和效率。
